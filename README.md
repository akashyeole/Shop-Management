# Shop-Management
A simple implementation of Shop Management System using Python 3.7 and SQLite Database connectivity. 
Design of program:
  1) Admin Login:
    a) View all sales
    b) View available products
    c) Add new product
    d) Update product price
    e) View employees
    f) Add employees
    g) Change password
    h) Logout

  2) Cashier Login:
    a) View avaliable products
    b) Store bill
    c) Change Password
    d) Logout
